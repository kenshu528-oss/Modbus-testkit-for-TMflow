========================================
tmflow_modbus_testkit.py 功能說明
========================================

這是什麼？
  Modbus Testkit for TMflow
  一個圖形化介面（GUI）的 Modbus 測試工具
  用來連線到 TMflow 程式，讀寫 Modbus 資料

========================================
主要功能
========================================

1. 連線管理
   ✓ 輸入 IP 和 Port 連線到 TMflow
   ✓ 顯示連線狀態
   ✓ 斷線功能

2. 座標讀取
   ✓ Base 座標 (7001-7012)
     - X, Y, Z 位置 (mm)
     - Rx, Ry, Rz 旋轉角度 (度)
   
   ✓ Tool 座標 (7025-7036)
     - X, Y, Z 位置 (mm)
     - Rx, Ry, Rz 旋轉角度 (度)
   
   ✓ Joint 角度 (7013-7024)
     - Joint 1-6 的角度 (度)

3. 狀態監控
   ✓ Robot Link - 機器人連線狀態
   ✓ Error - 錯誤狀態
   ✓ Project Running - 專案運行狀態
   ✓ ESTOP - 緊急停止狀態
   ✓ Robot State - 機器人狀態
   ✓ Operation Mode - 操作模式

4. User Define Area (9000-9999)
   ✓ 讀取自定義區域的資料
   ✓ 寫入自定義區域的資料
   ✓ 讀寫驗證

5. 性能測試
   ✓ 測試反應時間
   ✓ 測試成功率
   ✓ 可設定測試次數和間隔
   ✓ 生成測試報告

6. 自定義測試
   ✓ 自己指定要讀取的位址
   ✓ 自己指定資料型別
   ✓ 支援多種 Modbus 功能碼

7. 日誌管理
   ✓ 即時顯示操作記錄
   ✓ 儲存日誌到檔案
   ✓ 清除日誌

========================================
使用方式
========================================

1. 啟動程式
   雙擊「啟動測試工具.bat」

2. 連線到 TMflow
   - 輸入 TMflow 的 IP（例如：192.168.1.100）
   - 輸入 Port（通常是 502）
   - 點擊「連線」按鈕

3. 開始測試
   - 點擊「Base 座標」讀取 Base 座標
   - 點擊「Tool 座標」讀取 Tool 座標
   - 點擊「Joint 角度」讀取關節角度
   - 點擊「Robot 狀態」讀取機器人狀態

4. 查看結果
   - 所有結果會顯示在下方的日誌區域
   - 可以儲存日誌到檔案

========================================
實際應用
========================================

用途 1：監控機器人位置
  - 連線到 TMflow
  - 點擊「Base 座標」
  - 查看當前機器人的 X, Y, Z 位置

用途 2：檢查機器人狀態
  - 連線到 TMflow
  - 點擊「Robot 狀態」
  - 查看是否有錯誤、是否連線正常

用途 3：讀寫自定義資料
  - 連線到 TMflow
  - 點擊「User Define」
  - 可以讀取或寫入 9000-9999 位址的資料

用途 4：性能測試
  - 測試 Modbus 通訊的速度
  - 測試穩定性
  - 生成測試報告

========================================
支援的 Modbus 位址
========================================

座標資料（Input Registers, 功能碼 04）：
  7001-7012  Base 座標 (X,Y,Z,Rx,Ry,Rz)
  7013-7024  Joint 角度 (J1-J6)
  7025-7036  Tool 座標 (X,Y,Z,Rx,Ry,Rz)

狀態資料（Discrete Inputs, 功能碼 02）：
  7200  Robot Link
  7201  Error
  7202  Project Running
  7208  ESTOP

狀態資料（Input Registers, 功能碼 04）：
  7215  Robot State
  7216  Operation Mode

自定義區域（Holding Registers, 功能碼 03/06）：
  9000-9999  User Define Area（可讀寫）

========================================
簡單來說
========================================

這個程式就是一個「Modbus 讀寫工具」：
  - 連線到 TMflow
  - 讀取機器人的座標、狀態
  - 讀寫自定義資料
  - 測試通訊性能

不需要寫程式，用圖形介面就能操作！
