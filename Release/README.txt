TMflow Modbus 測試工具 v1.0.1.0002
獨立執行版（免安裝）
========================================

使用方式:

1. 雙擊「TMflow_Modbus_Testkit.exe」啟動程式
2. 輸入 TMflow 的 IP 位址和 Port（預設 502）
3. 點擊「連線」按鈕
4. 開始測試

特點:
✅ 不需要安裝 Python
✅ 不需要安裝任何套件
✅ 可以直接執行
✅ 完全獨立運行
