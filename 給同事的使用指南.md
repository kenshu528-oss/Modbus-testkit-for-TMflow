# TMflow Modbus 測試工具 - 使用指南

## 📦 取得方式

### 方式 1: 從 GitHub 下載（推薦）

1. 前往：https://github.com/kenshu528-oss/Modbus-testkit-for-TMflow
2. 點擊綠色的 **Code** 按鈕
3. 選擇 **Download ZIP**
4. 解壓縮到任意資料夾

### 方式 2: 使用 Git Clone

```bash
git clone https://github.com/kenshu528-oss/Modbus-testkit-for-TMflow.git
```

---

## 🔧 安裝步驟

### 前置需求

- **Python 3.7 或更高版本**
  - 檢查是否已安裝：開啟 CMD 輸入 `python --version`
  - 如果沒有安裝，請至 https://www.python.org/downloads/ 下載

### 安裝依賴

1. 開啟 CMD 或 PowerShell
2. 切換到專案資料夾：
   ```bash
   cd "解壓縮的資料夾路徑"
   ```
3. 安裝依賴套件：
   ```bash
   pip install -r requirements.txt
   ```

---

## 🚀 啟動程式

### 方式 1: 使用啟動腳本（最簡單）

雙擊 **啟動測試工具.bat**

### 方式 2: 手動啟動

```bash
python tmflow_modbus_testkit.py
```

## 📖 使用說明

### 1. 連線到 TMflow

1. 確保 TMflow 程式正在運行
2. 在測試工具中輸入 TMflow 的 IP 位址（例如：192.168.1.100）
3. 輸入 Port（預設：502）
4. 點擊「連線」按鈕

### 2. 執行測試

**預設測試**：
- 點擊對應的測試按鈕（Base 座標、Tool 座標等）

**自定義測試**：
- 在右側區域設定測試參數
- 點擊「執行自定義測試」

**性能測試**：
- 選擇測試類型和參數
- 點擊「開始測試」
- 測試完成後可生成報告

### 3. 查看結果

- 所有測試結果會顯示在日誌區域
- 可以點擊「儲存日誌」保存測試記錄

## 🔧 常見問題

### Q: 無法啟動程式？

**A**: 確認 Python 已安裝並執行：
```bash
pip install -r requirements.txt
```

### Q: 無法連線到 TMflow？

**A**: 
1. 確認 TMflow 程式正在運行
2. 確認 TMflow 的 Modbus TCP Server 已啟用
3. 檢查 IP 位址是否正確
4. 檢查防火牆設定（Port 502）

### Q: 缺少 pymodbus 模組？

**A**: 執行：
```bash
pip install pymodbus
```
