# 如何分享給同事

## 🎯 三種分享方式

---

## 方式 1: 使用發布包（最簡單）✨

### 步驟

1. **建立發布包**
   ```
   雙擊「建立發布包.bat」
   ```
   會自動產生：`TMflow_Modbus_Testkit_v1.0.1.0001.zip`

2. **分享 ZIP 檔案**
   - 透過電子郵件
   - 放在共享資料夾
   - 上傳到公司內部網站

3. **同事使用步驟**
   ```
   1. 解壓縮 ZIP 檔案
   2. 雙擊「快速安裝.bat」
   3. 雙擊「啟動測試工具.bat」
   ```

### 優點
- ✅ 最簡單，一個檔案搞定
- ✅ 包含所有必要檔案
- ✅ 有完整的使用說明

---

## 方式 2: 分享 GitHub 連結（最專業）

### 步驟

1. **提供連結**
   ```
   https://github.com/kenshu528-oss/Modbus-testkit-for-TMflow
   ```

2. **同事使用步驟**
   ```
   1. 點擊 Code > Download ZIP
   2. 解壓縮
   3. 執行 pip install -r requirements.txt
   4. 雙擊「啟動測試工具.bat」
   ```

### 優點
- ✅ 永遠是最新版本
- ✅ 可以看到更新歷史
- ✅ 可以回報問題（Issues）

---

## 方式 3: 內部網路共享（最方便）

### 步驟

1. **放在共享資料夾**
   ```
   例如: \\server\tools\TMflow_Modbus_Testkit
   ```

2. **建立使用說明**
   ```
   1. 複製整個資料夾到本機
   2. 雙擊「快速安裝.bat」
   3. 雙擊「啟動測試工具.bat」
   ```

### 優點
- ✅ 公司內部快速存取
- ✅ 統一版本管理
- ✅ 不需要外部網路

---

## 📋 給同事的簡易說明範本

你可以複製以下內容給同事：

```
【TMflow Modbus 測試工具】

這是用來測試 TMflow Modbus 功能的工具。

取得方式：
1. 從 GitHub 下載：https://github.com/kenshu528-oss/Modbus-testkit-for-TMflow
   或
2. 從共享資料夾取得 ZIP 檔案

使用步驟：
1. 解壓縮檔案
2. 雙擊「快速安裝.bat」（第一次使用）
3. 雙擊「啟動測試工具.bat」
4. 輸入 TMflow 的 IP 和 Port
5. 點擊「連線」開始測試

需要 Python 3.7+（如果沒有，請先安裝）

詳細說明請參考「給同事的使用指南.md」
```

---

## 🔧 前置需求檢查

提醒同事確認：
- ✅ 已安裝 Python 3.7 或更高版本
- ✅ 可以連線到 TMflow 的網路
- ✅ TMflow 的 Modbus TCP Server 已啟用

---

## 💡 建議

1. **第一次分享**：使用方式 1（發布包），最簡單
2. **長期使用**：教同事使用方式 2（GitHub），可以隨時更新
3. **公司內部**：使用方式 3（共享資料夾），最方便

---

## 🆘 如果同事遇到問題

常見問題解決：
1. **Python 未安裝**：提供 Python 安裝連結
2. **無法連線**：檢查 IP、Port、防火牆
3. **缺少套件**：執行 `pip install -r requirements.txt`

可以請他們到 GitHub Issues 回報問題：
https://github.com/kenshu528-oss/Modbus-testkit-for-TMflow/issues
